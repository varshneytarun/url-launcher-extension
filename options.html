Pattern Examples & Tips

  • Pattern: Use regular expressions (without delimiters). Example: INC\\d{7,} matches INC followed by 7+ digits
  • URL Template: Use {value} as placeholder. Example: https://example.com/ticket/{value}
  • ServiceNow Incident: Pattern: INC\\d{7,} ‚Üí URL: https://your-instance.service-now.com/nav_to.do?uri=incident.do?sysparm_query=number={value}
  • Jira: Pattern: [A-Z]{2,}-\\d+ ‚Üí URL: https://your-domain.atlassian.net/browse/{value}
  • GitHub Issue: Pattern: #\\d+ ‚Üí URL: https://github.com/owner/repo/issues/{value}
